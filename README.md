OutFit-Me

Overview

OutFit-Me is a Python application that recommends outfits based on temperature, weather conditions, occasion, and personal temperature preference. The project was created to strengthen Python programming skills while practicing functions, conditional statements, dictionaries, input validation, and application design.

Features

* Accepts user input for temperature
* Accepts weather conditions (sunny, rain, snow, or windy)
* Recommends outfits based on the current temperature
* Provides recommendations for different occasions (casual, work, or gym)
* Suggests weather-specific accessories such as umbrellas, sunglasses, scarves, and gloves
* Allows users to choose whether they typically prefer warmer or cooler clothing
* Validates numeric temperature input
* Uses modular functions to organize application logic

How It Works

The application asks the user to enter:

1. The current temperature
2. The weather condition
3. The occasion
4. Their personal temperature preference

Based on those inputs, the application generates a customized outfit recommendation.

Example

Temperature (°F): 72
Weather: rain
Occasion: work
Do you usually get warm or cold? warm
Recommended Outfit
-------------------------
Button-down shirt, slacks, and loafers
• Umbrella
• Rain jacket
