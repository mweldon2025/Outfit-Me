OutFit-Me

Overview

OutFit-Me is a Python application that recommends outfits based on temperature, weather conditions, occasion, and personal temperature preference. The project was created to strengthen Python programming skills while practicing functions, conditional statements, dictionaries, input validation, and modular application design.

Features

* Accepts user input for temperature
* Accepts weather conditions (sunny, rain, snow, or windy)
* Recommends outfits based on the current temperature
* Provides recommendations for different occasions (casual, work, or gym)
* Suggests weather-specific accessories based on current weather conditions
* Allows users to specify whether they prefer warmer or cooler clothing
* Validates numeric temperature input
* Uses modular functions to organize application logic

How It Works

The application prompts the user to enter:

1. The current temperature
2. The weather condition
3. The occasion
4. Their personal temperature preference

Using those inputs, the application:

* Categorizes the temperature.
* Selects an outfit based on the temperature and occasion.
* Recommends weather-specific accessories.
* Provides an additional recommendation based on the user’s clothing preference.
